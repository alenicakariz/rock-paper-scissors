# rock-paper-scissors

Site Link: https://alenicakariz.github.io/rock-paper-scissors/

A classic **Rock, Paper, Scissors** game I built to practice JavaScript fundamentals.

Players can choose between the three options, and the computer generates a random move to determine the outcome. 
This game also keeps track of wins, losses, and ties through local storage.

This project demonstrates essential concepts such as conditional logic, random number generation, event handling, local storage, and dynamic DOM updates to display results instantly.
